# Phase 2: 解析

スタッフの確認OKを受けて「解析を開始します」と宣言してから解析に入る。

## 2.1 睡眠指標（8項目を算出）

### (1) 平均睡眠時間
- 全日の睡眠時間の平均を算出（h:mm形式）
- 理想値は年齢別レンジ（コンサルティングシートから年代を読み取る）：
  10代=8-10h, 20代=7-9h, 30代=7-8.5h, 40代=7-8h, 50代=6.5-7.5h, 60代=6-7h, 70代以上=5.5-7h

### (2) 寝つき平均時間
優先順位：
- A) 画面に「入眠時間: 00:xx」と明記 → そのまま採用（最優先）
- B) 入眠までの赤線/入眠開始が読み取れる → 可能なら推定
- C) 上記不可 → 寝つき評価(1〜5)から推定マッピング：
  - 5: 0〜10分 / 4: 10〜20分 / 3: 20〜30分 / 2: 30〜45分 / 1: 45分以上
  - ※推定時は必ず「推定」と表記
- 理想値：10分前後（固定）
- 5分以内が多い場合：「睡眠負債の可能性があります」コメントを添える（断定しない）

### (3) 寝返り回数平均
- 全日の平均を算出
- 適正：20〜30回（固定）

### (4) 活動量平均
- 1〜5スケールの全日平均
- 理想：3以上

### (5) 覚醒回数分布
- 0〜2回 / 3〜5回 / 5回以上の3カテゴリ
- 各カテゴリの日数と全体に対する割合(%)を算出

### (6) 就寝中の向き
- 仰向け / 右向き / 左向き / うつ伏せ の4分類
- 各分類の%(睡眠中=グレー塗り範囲のみで集計)

### (7) 入眠の体の向き
- 各日の入眠時の向きを集計し、最も多い向きを特定
- 代表的な遷移パターン（例：「仰向けで入眠→途中から右向き」）を短文で記述

### (8) ソーシャルジェットラグ
算出手順：
1. 全日の入眠時刻・起床時刻・睡眠時間の中央値を算出
2. 各日について中央値からのズレ(分)を算出：
   ズレ = max(|入眠-中央値|, |起床-中央値|, |睡眠時間-中央値|)
3. 最大ズレの日を採用し、その値をジェットラグとして表示
4. ズレの主因（遅寝/遅起き/長寝/短寝）を短文で説明
※休日/平日の区分は使用しない

## 2.2 総合スコア算出（/100点）

| 項目 | 配点 | 算出方法 |
|---|---|---|
| 睡眠時間充足度 | 20点 | 理想時間範囲内=20, 30分不足ごとに-4, 1h以上不足=8以下 |
| 寝つき | 15点 | 10分=15, 5-15分=12, 15-20分=10, 20-30分=7, 30分以上=3 |
| リズム安定性 | 20点 | ジェットラグ0-30分=20, 30-60分=16, 1-2h=12, 2-3h=8, 3h以上=4 |
| 覚醒の少なさ | 15点 | 0-2回割合×15（例：80%なら12点） |
| 活動量 | 10点 | 平均3以上=10, 2.5-3=7, 2-2.5=5, 2未満=3 |
| 生活習慣 | 10点 | (54-点数)/54×10（低いほど良いので反転） |
| ストレス | 10点 | PSS0-13=10, 14-19=7, 20-26=5, 27以上=2 |

## 2.3 PSS（Perceived Stress Scale）

- リスク判定（総合点）：0〜13=低リスク / 14〜26=中ストレス / 27以上=高ストレス
- 表示項目：ネガティブ（受動的）合計、ポジティブ（能動的）合計、総合点、リスク
- コメント：睡眠指標との"つながり"を1〜2行で説明

## 2.4 生活習慣チェック（54点版）

- 3段階評価で「低いほど良い」
- カテゴリ配点：睡眠環境15 / 食事・飲料15 / 運動9 / ストレス管理6 / 就寝・起床9（合計54）
- 評価レンジ：A=19〜28（良好）/ B=29〜38（やや問題あり）/ C=39〜54（改善で向上が見込める）

## 2.5 バランス評価（★4段階）

各指標を★1〜4で評価する：
- 睡眠時間：理想範囲内=★4, やや不足(30分以内)=★3, 不足(30-60分)=★2, 大幅不足(60分以上)=★1
- 寝つき：10分以内=★4, 10-20分=★3, 20-30分=★2, 30分以上=★1
- リズム安定：ジェットラグ30分以内=★4, 1h以内=★3, 2h以内=★2, 2h超=★1
- 覚醒：0-2回80%以上=★4, 60%以上=★3, 40%以上=★2, 40%未満=★1
- 活動量：3以上=★4, 2.5-3=★3, 2-2.5=★2, 2未満=★1
- 睡眠姿勢：仰向け60%以上=★4, 40-60%=★3, 仰向け40%未満=★2, うつ伏せ多=★1

★表示形式：★=&#9733; ☆=&#9734;（HTMLエンティティ）

## 2.6 総評と提案の生成

### 総評の統合順序
「睡眠・生活習慣・ストレス・ヒアリング主訴」を統合して作成する。
優先順：①睡眠分析結果 → ②生活習慣改善 → ③ストレス改善 → ④睡眠環境改善 → ⑤寝具提案

### 提案の構成
- **最優先3つ**（メインレポート1枚目）：今すぐ・効果大・主訴に刺さるもの。各提案に「具体的行動」＋「理由1行」
- **追加候補5つ**（メインレポート2枚目）：できそうな順。「いつ・どれくらい・何を」で記述
- **おすすめ商品2つ**：寝具 or スリープテックに限定。商品カテゴリ案＋コメント下書き（画像と最終コメントはスタッフが入れる前提）

### 矛盾時の対応
主訴と数値が矛盾 → 主訴を先に扱い、「数値的にはここも気になるかもしれません」と併記

## 2.7 前処理ルール

- 欠損日は除外し「欠損◯日」と注記
- 体の向きは睡眠中（グレー塗り範囲）のみ集計
- 時刻の日付跨ぎは自動補正
- 算出不可項目は「算出不可（理由）」と明記

## 3.3 詳細版コンテンツガイドライン

### 全体トーン
- お客様が自宅で一人で読んでも理解できる平易さ
- スタッフが「ここ」と指差しして抜粋説明しやすいセクション構成
- 専門用語には必ず（ ）で言い換えを添える

### セクション1: 睡眠データ詳細
- まず「全体の傾向」を2〜3行で述べる
- 日別データテーブル（日付/入眠/起床/睡眠時間/寝つき/覚醒/活動量/寝返り）を表示
- 特徴的な日があれば「○日は特に〜」と補足

### セクション2: 入眠時の体の向き
- 最も多い入眠姿勢とその割合
- 途中での姿勢変化パターン
- 姿勢が睡眠の質にどう影響しうるかを1〜2行で解説（断定しない）

### セクション3: ソーシャルジェットラグ詳細
- ジェットラグとは何かを日常の例え（時差ボケのようなもの）で1行説明
- 日別のズレ一覧テーブル
- 最もズレが大きかった日の分析（何が原因でズレたか）
- リズムが安定している日との比較

### セクション4: ストレスと睡眠の関連
- PSSの結果と睡眠データの関連を具体的に説明
- 「ストレスが高いと寝つきが悪くなったり、覚醒が増えたりすることがあります」等の一般知識
- その人のデータでどう表れているかを紐づけ

### セクション5: 生活習慣カテゴリ別アドバイス
- 5カテゴリそれぞれについて：
  - 現在の点数と評価
  - 具体的な改善アドバイス（1〜2個）
  - 「なぜそれが睡眠に影響するか」の一言解説

### セクション6: 改善提案詳細
- 最優先3つ＋追加5つ、計8つの提案を詳細に展開
- 各提案ごとに：提案内容 / なぜ効果があるか / 具体的なやり方（いつ・どれくらい・何を）

### セクション7: 用語集
以下の用語を固定で含める（データに応じて追加可）：
- 総合スコア：8つの指標を100点満点で総合評価したもの
- PSS（知覚されたストレス尺度）：日常のストレスの感じ方を数値化するアンケート
- ソーシャルジェットラグ：日によって寝る時間・起きる時間がバラバラになることで体内時計がズレる現象。時差ボケのようなもの
- 活動量スコア：日中の体の動きの多さを1〜5で表した数値
- 覚醒：睡眠中に一時的に目が覚めること（中途覚醒）
- 寝返り：睡眠中に無意識に体の向きを変えること。適度な寝返りは血行促進や体圧分散に必要
- 体圧分散：体の重さが一箇所に集中せず、マットレス全体に均等にかかること
