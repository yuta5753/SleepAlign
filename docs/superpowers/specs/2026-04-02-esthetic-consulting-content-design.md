# エステコンサル コンテンツ魅力・フック強化 企画書

**作成日**: 2026-04-02
**ステータス**: リサーチ検証済み
**用途**: 内部企画書（Webページ・提案書の元ネタとして使用）

---

## 1. 企画の目的

エステコンサルサービスの「入口の訴求（フック）」と「メニューの打ち出し方」を明確にし、以下に繋げる：

- ビジネスパートナー・辻が見込み客に**何をどう話すか**が固まる
- サービスメニューの**構成と見せ方**が整理される
- 上記が固まることで、Webページ（LP）・提案書を迷わず作れるようになる

## 2. サービスコンセプトとポジショニング

### コンセプトの3層構造

| 層 | 内容 |
|---|---|
| **入口（フック）** | 売上が上がる・物販が売れる・集客に繋がる |
| **本題（コア）** | 顧客体験（CX）を「仕組み」として設計する |
| **手段（How）** | CX導線設計 + スタッフ教育 + マニュアル化（+ オプションでシステム・HP・集客設計） |

### ひと言で言うと

> **「売上が上がる"顧客体験の仕組み"を、あなたのサロンに作る」**

### 他のサロンコンサルとの違い

| | 集客コンサル | 技術指導 | 経営コンサル | **辻のサービス** |
|---|---|---|---|---|
| やること | 広告・SNS運用 | 施術スキルUP | 数字管理・戦略 | **顧客体験の仕組み化** |
| 効果の持続性 | 広告止めたら終わり | その人限り | 経営者依存 | **仕組みとして残る** |
| スタッフへの展開 | △ | 個人の力量次第 | △ | **誰でも再現できる設計** |
| 対象 | 集客だけ（点） | 技術だけ（点） | 経営全体（広すぎ） | **顧客体験の導線全体（線）** |

### ポジショニングの核

**「ノウハウを教える人」ではなく「仕組みを作る人」**

- 他のコンサルは「こうしたらいいですよ」と教える → オーナー個人の力量に依存
- 辻のサービスは「この仕組みを入れれば回る」を設計・実装する → 誰がやっても成果が出る

### ポジショニングの裏付け（リサーチ結果）

**「CX設計×仕組み化」は競合ほぼゼロの空白地帯。**

- 「顧客体験の設計」をエステサロン向けに打ち出しているプレイヤーは国内でほぼ存在しない
- CX設計は大企業向けコンサルファーム（デロイト・IBM等）の領域で、サロン市場には降りてきていない
- b-modelsの「カウンセリング再現性向上」が唯一近いが、カウンセリング局面のみで顧客体験「全体」ではない

```
【ポジショニングマップ】

縦軸：介入範囲（点 ↔ 面）
横軸：対象フェーズ（新規集客 ↔ 既存顧客のLTV最大化）

・SNS/集客コンサル        → 左下（新規集客の点施策）
・船井総研/フューチャーブレーン → 中央（幅広いが高価格・大企業向け）
・b-models              → 中央やや右（カウンセリング設計は強いが体験全体ではない）
・辻のポジション           → 右上（既存顧客×体験全体の面設計）★競合ゼロに近い
```

### もう一つの差別化：「学んでも実行できない」問題を解決する

リサーチで判明した重要な構造的問題：

- オーナーの63%が「セミナー・スクール」で経営を学んでいる
- しかし業界識者は「学んでも実行できていない」と繰り返し指摘
- **辻のサービスは「教える」のではなく「設計して実行まで伴走する」** → セミナーとの明確な差別化

## 3. フック設計 — 見込み客の関心を掴む「入口の言葉」

### フックの前提

見込み客（エステサロンオーナー）はリピート率やLTVなどの指標を持っていない。つまり「課題を自覚していない」状態。だから**「あなたの課題はこれですよ」ではなく、「こういう成果が出ますよ」で入る**。

**リサーチで裏付けられたファクト：**
- 「うまくいっている」と回答するオーナーが約9割（prtimes.jp調査）
- 一方でエステサロンの平均リピート率はわずか21%（ホットペッパービューティー調査）、理想は70-80%
- この矛盾 = 課題を自覚していない証拠。「課題指摘型」ではなく「成果提示型」が正解

### メインフック

> **「新規を20人集めるのと、5人の新規で同じ売上が上がるのと、どっちがいいですか？」**

このフックが強い理由：
1. **比較で見せている** — 数字の対比は抽象論より圧倒的に刺さる
2. **「集客しなくていい」ではなく「少ない集客で同じ成果」** — 集客を否定しないから反発が生まれない
3. **自然にCX設計の話に繋がる** — 「なぜ5人で同じ売上？」→「1人あたりのリピート・物販・単価が上がるから」→「それが顧客体験の仕組み化です」
4. **競合にない表現形式** — 競合は「データ比較」が多い中、「選択肢を突きつける」形式で差別化できている

### フックの橋渡し（ファクトで裏付ける）

フック単体では「なぜ5人で同じ売上？」が伝わりきらない場合がある。以下のファクトで橋渡しする：

- **「新規1人を集めるコストは、既存客を維持するコストの5倍」**（業界で広く認知されているファクト）
- **「リピート率が30%→60%に上がると、年間売上に約380万円の差が生まれる」**（b-merit調査）

### フックから本題への接続（会話の流れ・改良版）

```
「新規20人と、新規5人で同じ売上。どっちがいいですか？」
  ↓
「実は、新規を1人集めるコストは、既存のお客様に来てもらうコストの5倍かかると言われています」
  ↓
「つまり、今のお客様が自然にリピートして、物販も買ってくれる"流れ"を作れば、
 少ない新規でも売上は変わらない——むしろ上がることも多い」
  ↓
「じゃあどうやるか。カウンセリングから施術、ホームケア、次回予約まで、
 お客様が"また来たい""これ欲しい"と思う体験を設計するんです」
  ↓
「それを仕組みにするから、スタッフ誰がやっても同じ成果が出ます」
```

### サブフック（関心別）

| オーナーの関心 | フック |
|---|---|
| 物販が売れない | 「売り込まなくても、お客様が"欲しい"と言う流れがあります」 |
| スタッフに依存 | 「あなた以外のスタッフでも、同じようにリピート・物販が取れる仕組みがあります」 |
| 漠然とした不安 | 「うちは大丈夫、と思っていても、エステサロンの平均リピート率は実は21%。ほとんどのお客様は2回目に来ていません」 |

### 表現上の注意

- **「仕組み化」を単体で使わない。** 業界で使われすぎて差別化が弱い。「リピートの仕組み」「物販が売れる流れ」のように目的語をつける
- **「CX設計」「LTV」「カスタマージャーニー」は見込み客に使わない。** 大企業向けコンサル用語であり、サロンオーナーには伝わりにくい
- **「誰がやっても同じ成果が出るやり方を作る」** が「仕組み化」の最も伝わりやすい言い換え

## 4. サービス構成 — メニューの打ち出し方

### 基本方針

サービスメニューを「CX設計」「LTV設計」という専門用語で見せない。見込み客の「欲しい成果」の言葉でパッケージ名をつける。

### メインサービス（コア）

**サービス名（仮）：「売れる仕組みづくり」**

| ステップ | 内容 | 見込み客への見せ方 |
|---|---|---|
| Step1：現状診断 | 顧客体験の棚卸し + コンセプト・ターゲットの現状確認 | 「今の"お客様の流れ"と"サロンの軸"を見える化します」 |
| Step2：コンセプト再定義 | サロンの「誰に・何を・なぜ」を固め直す | 「仕組みの土台になる"サロンの軸"を一緒に作ります」 |
| Step3：導線設計 | コンセプトに基づいたCX導線設計（リピート・物販・単価UPが自然に生まれる体験の流れ） | 「少ない新規でも売上が上がる"流れ"を設計します」 |
| Step4：仕組み化 | マニュアル・トークスクリプト・スタッフ教育の整備 | 「誰がやっても同じ成果が出る仕組みに落とし込みます」 |
| Step5：定着支援 | 導入後のフォロー・改善 | 「現場で回るまで伴走します」 |

**ステップの思想**: コンセプト → 導線 → 仕組み の順番で、「軸を決める → 体験を設計する → 再現できるようにする」というストーリー。

**「セミナーとの違い」を明確に：** セミナーは「知識を教える」で終わり、実行はオーナー次第。辻のサービスはStep3〜5で「設計→仕組み化→定着まで伴走」するから、学んだだけで終わらない。

### オプションサービス（付加価値）

| オプション | 内容 | 見せ方のポイント |
|---|---|---|
| 予約管理システム構築 | バイブコーディングで自社用に作成 | 「外注すれば数十万円かかるシステムが、仕組みの一部としてついてきます」 |
| 自社HP作成 | 同上 | 同上 |
| 集客施策の設計 | コンセプトに基づく集客導線の策定 | 「仕組みができた上での集客だから、効率が全然違います」 |

### オプションの見せ方の戦略

**「本来は別途費用がかかるものが、仕組みの一環としてついてくる」** というお得感。ただし、あくまでメインサービスの価値を高めるための付加であり、オプション単体が売りにならないようにする。

### 価格設計（方向性）

- **メインサービス**: プロジェクト型の一括 or 分割（Step1だけ受けて様子を見るのもOK）
- **オプション**: メインとセットで提供。単体販売は原則しない
- **Step1を入口に**: Step1（現状診断）だけでも価値があり、結果を見て次に進むか判断できる設計

**価格帯の参考情報（リサーチ結果）：**
- 大手コンサル（船井総研・フューチャーブレーン等）：月額15〜25万円
- 個人系集客コンサル（SNS系等）：月額3〜10万円
- **月額8〜15万円帯に空白がある** — 「個人コンサル×CX設計専門」として未開拓の価格帯

## 5. 語り方の設計 — 見込み客との会話の型

### 第1段階：関心を掴む（フック）

相手の関心に合わせてメインフックを使い分ける：

| 相手の状況 | 切り出し方 |
|---|---|
| 集客・売上に関心 | 「新規を20人集めるのと、5人で同じ売上上がるのと、どっちがいいですか？」 |
| 物販が売れない | 「売り込まなくても、お客様から"欲しい"と言われる流れって作れるんですよ」 |
| スタッフに依存 | 「オーナーじゃなくても、同じようにリピートと物販が取れる方法があります」 |

### 第2段階：ファクトで橋渡し → 本題に繋げる

```
「なぜそれができるかというと——」
  ↓
「実は、新規を1人集めるコストは既存客の5倍。
 それなのに多くのサロンはリピート率が21%、8割のお客様は2回目に来ていない」
  ↓
「じゃあ何が足りないか。
 お客様がサロンに来てから帰るまでの"体験の流れ"が設計されていないんです」
  ↓
「カウンセリングで何を聞くか、施術中に何を伝えるか、
 ホームケアをどう提案するか、次回予約をどう取るか」
  ↓
「この流れが設計されていれば、リピートも物販も"自然に"生まれる」
  ↓
「しかもそれを仕組みにするから、スタッフ誰がやっても同じ成果が出る」
```

### 第3段階：サービスの説明（具体的に何をするか）

```
「具体的には、まずサロンの今の状態を一緒に見える化します」
  ↓
「その上で、サロンの"誰に何を届けるか"という軸を固め直して」
  ↓
「その軸に沿った体験の流れを設計する」
  ↓
「最後にそれをマニュアルやスクリプトに落とし込んで、
 スタッフが迷わず動ける状態にします」
  ↓
「必要なら予約システムやHPも一緒に作れるので、
 本来は別々に外注するものが全部まとめて仕組みになります」
  ↓
「セミナーで学んでも結局やりきれない、という経験ありませんか？
 うちは"作って・落とし込んで・回るまで伴走する"ので、そこが全然違います」
```

### 紹介者（第三者）が使えるひと言

> **「サロンの"お客様の体験"を丸ごと設計してくれる人がいるよ。物販もリピートもスタッフ全員できるようになる仕組みを作ってくれる」**

## 6. 下流展開への接続（Webページ・提案書への橋渡し）

この企画書をもとに、以下の順で展開する：

### Webページ（LP・サービス紹介）

- **構成**: メインフック → ファクト（リピート率21%・5倍コスト差）→ 見込み客の悩み共感 → サービス概要（5ステップ） → 他との違い → オプション紹介
- **トーン**: フックと語り方設計の言葉をそのまま使う（専門用語は使わない）
- **CTA**: Step1（現状診断）への申込み or 無料相談

### 提案書

- **構成**: 相手の状況への共感 → メインフック → ファクトで裏付け → サービスの流れ（5ステップ） → オプション → 価格
- **カスタマイズポイント**: Step1の前に簡易ヒアリングを行い、相手の状況に合わせた「フック」と「期待される成果」を冒頭に入れる

## 7. 前提と制約

### 現在の営業チャネル
- ビジネスパートナー兼営業担当による直接営業・紹介が中心
- フックが効くべき場面は「初回の会話・説明の中で」と「紹介者が第三者に説明する時」

### 見込み客の特徴
- リピート率・LTV・物販率などの指標を持っていない
- 課題を自覚していないため、「課題指摘型」ではなく「成果提示型」でアプローチする
- 「集客」「物販が売れる」「売上が上がる」に強く反応する
- 複数スタッフのサロンでは「自分以外でもできるようになる」にも反応する
- 63%がセミナーで学んでいるが、実行に落とし込めていない

### サービスの本質的な差別化
- **仕組み化が核**: ノウハウを教えるのではなく、仕組みを作る
- **CX設計という切り口**: 顧客体験の導線全体を設計するのは競合にほぼいない（リサーチで確認済み）
- **実行まで伴走**: セミナー型との明確な差別化
- **組織人事・実行伴走は手段**: 上記を実現するための付随サービス

### 飲食企業向けサービスとの一貫性
- 飲食側：「コスト削減（フック）→ 経営構造の再設計（本題）」
- エステ側：「売上UP・物販（フック）→ 顧客体験の仕組み化（本題）」
- 共通思想：**「入口は成果の言葉、本題は構造・仕組みの設計」**

## 8. リサーチデータ（根拠）

### 市場環境
- エステ市場規模：3,043億円（2024年度）、5年連続縮小
- 廃業率：3年以内に約60%が廃業（参考値。出典によりばらつきあり）
- 開業のハードルが低く新規参入が続く → 差別化できないサロンは淘汰される

### リピート・物販の実態
- エステサロン平均リピート率（6ヶ月）：**21%**（ホットペッパービューティー調査）
- 理想のリピート率：70〜80%（業界一般的な目標値）
- 物販が売上に占める割合：約33.7%（経済産業省データ）
- リピート率30%→60%で年間売上に約380万円の差（b-merit調査）
- 新規獲得コストは既存維持の5倍（業界共通認識）

### スタッフ関連
- 1年以内の離職率：約29%、3年目までで約56%（ritsubi.co.jp）
- 有効求人倍率：1.10〜1.20前後（2026年1月推定）

### オーナーの意識
- 「経営がうまくいっている」と回答：約9割（prtimes.jp調査）
- 「リピート率の担保・向上」が課題：64%が回答（同調査）
- 経営学習方法1位：セミナー・スクール（63%）

### 競合の価格帯
- 大手総合（船井総研等）：月額15〜25万円
- 中堅エステ特化（b-models等）：相談後個別設定
- 個人系集客コンサル：月額3〜10万円
- **空白帯：月額8〜15万円**
