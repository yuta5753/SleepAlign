# 飲食リモデリング — 資材制作プロジェクト キックオフプロンプト

## 使い方
新しい会話を開始し、以下のプロンプトを最初のメッセージとしてそのまま貼り付ける。

---

## プロンプト（ここから下をコピーして貼り付け）

```
[ロール: 仕組み構築]

飲食企業向け「経営構造リモデリング支援サービス」の営業資料・集客施策の資材制作を行う。
戦略設計は完了済み。エージェントチームを編成して、精度の高い資材を制作したい。

【必ず最初に読むファイル】
1. 実行ブリーフ: /Users/tsujiyuuta/SleepAlign/docs/superpowers/specs/2026-04-02-food-service-execution-brief.md
   → 制作する資材のリスト・仕様・デザイン指針・メッセージング指針・使用データ一覧がすべてまとまっている

2. 設計ドキュメント: /Users/tsujiyuuta/SleepAlign/docs/superpowers/specs/2026-04-02-food-service-remodeling-design.md
   → サービスの全体設計・ターゲット・競合分析・営業戦略の詳細

【成果物の保存先】
~/Desktop/claude-outputs/04_店舗ビジネス支援/飲食リモデリング/営業ツール/

【制作済み資材（初版・品質改善が必要）】
- 01_アプローチ文面.md — カジュアルDM + フォーマルメールのテンプレート
- 02_課題ヒアリングシート.md — 60分面談の進行ガイド（PDF化が必要）
- 03_Phase1提案書.pdf — 1枚もの提案書（レイアウト・品質の改善が必要）

【進め方】
1. 上記2ファイルを読み、制作対象の資材と品質基準を把握する
2. 制作する資材に最適なエージェントチームの構成案を提案する
   - 各エージェントの役割・担当範囲・使用モデル（Opus/Sonnet）を明示
   - なぜその構成が最適かの理由も添える
3. 私の承認を得てからエージェントチームを起動し、制作に着手する
4. 各資材の完成後、品質レビューを行い、フィードバックを反映する

※ エージェントチーム構成案は必ず私に確認を取ってから進めること。勝手に着手しない。
```

## 補足: 個別資材だけ作りたい場合

特定の資材だけ制作したい場合は、上記プロンプトの【進め方】の前に以下を追加する。

### Phase 1提案書のみ
```
【今回の制作対象】
Phase 1提案書（実行ブリーフ§5-3）を高品質で作り直す。
既存版（03_Phase1提案書.pdf）の課題を特定し、デザイン指針（ブリーフ§4）に沿って再制作する。
```

### サービス概要資料のみ
```
【今回の制作対象】
サービス概要資料（実行ブリーフ§5-4）を制作する。
A4・3〜5ページ。面談後やセミナー後に送付する詳細説明資料。
```

### セミナースライドのみ
```
【今回の制作対象】
セミナースライド（実行ブリーフ§5-5）を制作する。
テーマ「3店舗の壁を超える — AI時代の飲食店経営の仕組みづくり」、90分構成。
設計ドキュメント§7のセミナー戦略に沿って制作する。
```

### 自己診断ツールのみ
```
【今回の制作対象】
属人化度チェックリスト（実行ブリーフ§5-6）を制作する。
セミナー内で使用する前提。1分で回答できる手軽さで、結果が「もっと知りたい」につながる設計。
```

### note記事のみ
```
【今回の制作対象】
note記事シリーズ（実行ブリーフ§5-7）を制作する。
「壁」シリーズとして、認知獲得・専門性発信が目的。noteブランドルール（CLAUDE.md参照）に準拠。
```
