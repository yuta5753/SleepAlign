# 競合調査レポート：エステサロン向けコンサルティング市場

**調査日**: 2026-04-02  
**調査者**: リサーチャー（Claude）  
**調査対象**: 日本国内のエステサロン向けコンサルティング市場

---

## 1. 競合プレイヤーの分類と代表例

### タイプA：大手総合コンサルティング会社（美容特化部門あり）

**船井総合研究所（船井総研）**
- 美容室・エステ経営.com を運営する日本最大級の美容業界特化コンサル
- 社員規模1,276名。全国のエステサロン・美容室を対象
- 支援領域：新規集客、リピート率向上、採用・育成・評価制度、多店舗展開、事業承継・M&A
- 「美容業界の新たなスタンダードを作る」を使命として掲げる
- 料金：月額15万円〜（相場情報。公式には非掲載）

**StockSun株式会社**
- 美容室・エステ向けのWeb・デジタルマーケティング特化コンサル
- 集客支援（SEO・MEO・SNS・広告）、採用支援（YouTube採用）、DX支援
- 「領域横断型」が強みで、多職能コンサルタントが在籍
- 料金：非公開（別途料金ページあり）

---

### タイプB：エステ業界特化の中堅コンサルティング会社

**株式会社b-models**
- 船井総研出身者が創業。100店舗超のエステサロン支援実績
- 4つのメニュー：集客/販促、採用/人材教育、契約率UP（カウンセリング再現性向上）、出店/ビジネス開発
- SLIMSHOCKシステム：物件・商品・技術・人材のワンストップ開業支援
- 「オーナー・スタッフ・お客様が幸せになれるコンサルティング」を訴求
- 料金：初回無料相談。費用は相談後に個別設定

**株式会社フューチャーブレーン**
- 約300社700店舗の支援実績
- 集客・教育・求人・メニュー開発を包括的に提供
- 自社でブライダルエステ・ヘッドスパ・岩盤ホットヨガスタジオを経営しており「現役経営者コンサル」を訴求
- 料金：月額25万円〜（スタンダードプラン）

**株式会社のぞみコンサルティング**
- 相談500件超、コンサルティング実績100件超
- 無料2時間経営相談を提供
- 6ヶ月で新規客9.7倍などの成果実績を前面に出す集客重視型
- 料金：月額10万円〜（エコノミーコース）〜25万円（顧問コンサル）

**株式会社NBS（clear-sp.com運営）**
- 開業支援・サポートが主軸
- LTV向上、スタッフ教育、月商アップ支援もカバー
- ブログで情報発信が充実（エステコンサルの費用相場、LTVの考え方など）

**株式会社スネイルペース**
- 自社運営サロンを複数展開しつつコンサルも提供
- スタッフ満足を重視した経営を訴求
- 料金：499,800円〜（6ヶ月短期プラン）

### タイプC：個人・小規模コンサルタント（実務家出身）

**きれいプロデュース（松下ひとみ）**
- 美容業界30年・店舗運営15年以上の現役オーナーが担当
- 「永遠にリピートし続けてくれる上顧客の作り方」に特化
- 開業半年で月商150万円、10年で1000万円の成功事例を保有
- エステ以外にネイル・まつエク・脱毛サロンも対応
- 料金：初回30分無料。以降は非公開

**キズカスカンパニー（seikou-salon.com）**

- 社員研修：半日10万円、1日15万円（交通費別）
- 初回相談：33,000円（税込・120分）

**幸せサロン育成塾**
- 個人サロン・自宅サロン向けの集客・リピート仕組み化
- LINEコンサル（月額4万円〜）などオンライン型が充実

**有限会社ヒューマンサポート**（大阪）
- 「感動マーケティング」による顧客関係構築を訴求
- 料金：経営相談 3万5,000円〜（2時間）

## 3. 訴求メッセージの傾向

### 「集客増加」訴求（最も多い・競争激化）
「新規客を増やす」「月商○○万円達成」「インスタで集客」など数字・成果主導の訴求。  
集客コンサル・SNSコンサル系がこのゾーンに密集しており、最も競争が激しい。

### 「リピート率向上」訴求（多い）
「リピート率を上げる」「リピーター獲得」「次回予約の仕組み」など定着化を訴求。  
船井総研・b-models・個人系コンサルの多くがサブ訴求として言及。ただし「なぜリピートするのか」の体験設計論まで踏み込んでいるプレイヤーは少ない。

### 「LTV向上・物販促進」訴求（少ない）
LTVという言葉は使われるが、「体系的な顧客生涯価値の設計」を前面に出しているプレイヤーは少数。  
b-modelsが「契約率UP＝カウンセリング設計」という形で近い概念を提供しているが、顧客体験全体のLTV設計には踏み込んでいない。

### 「スタッフ育成・人材教育」訴求（中程度）
採用・教育・マニュアル構築を提供する会社は多い。ただし「スタッフが誰でも同じ顧客体験を届けられる設計」という仕組みとしての提供は見当たらない（推測）。

### 「経営戦略・財務・組織」訴求（大手に多い）
船井総研・フューチャーブレーンなどが経営全般をカバー。個人コンサルはこの領域に踏み込みにくい。

### 「仕組み化・再現性」訴求（散見される）
「仕組みで解決」「再現性のある経営」という言葉は使われ始めているが、「顧客体験の仕組み化」として統合的に提供しているプレイヤーはいない。集客の仕組み化・スタッフ教育の仕組み化という部分的な使われ方に留まっている。

## 4. 「CX設計・仕組み化」の競合状況

### 現状の調査結果

「顧客体験（CX）設計」「カスタマージャーニー」という言葉でエステサロン向けコンサルを打ち出しているプレイヤーは、**ほぼ存在しない**。

- CX設計の概念を使っているのは博報堂コンサルティング・IBM・デロイト・アクセンチュアなど大企業向けのコンサルファームのみ
- エステサロン業界での「CX設計」「顧客体験を仕組みとして設計する」という切り口は確認されなかった
- b-modelsの「契約率UPコンサル（カウンセリング再現性向上）」が唯一、このコンセプトに近い領域に触れている
- 「サロン 顧客体験 再現性 誰でも コンサル 仕組み」という複合キーワードでの検索では、該当するサービスは見つからなかった

### 近接しているプレイヤーとの比較

| プレイヤー | 近い要素 | 欠けている要素 |
|---|---|---|
| b-models | カウンセリング再現性、契約率設計 | 顧客体験の「全体設計（来店前〜来店後）」視点が薄い |
| 幸せサロン育成塾 | 仕組み化の発信・LINEフォロー | 施術前後の体験設計に踏み込んでいない |
| きれいプロデュース | リピーター育成の設計視点 | 「誰でも再現できる」仕組み化の表現なし |
| 船井総研 | 組織・評価・マネジメントは強い | 個店レベルの顧客接点設計ではない |
| 幸せサロン育成塾 | 低価格・個人サロン特化 | CX・LTV・物販を統合した設計提供なし |

## 5. 差別化の機会

### 空白地帯の特定

調査の結果、以下の領域が**既存プレイヤーがカバーしていない空白地帯**として確認された。

**空白地帯①：顧客体験の全体設計（初回接触〜来店後）を仕組みとして提供するコンサル**  
現状の競合は「集客」か「リピート対策」かのどちらかに偏っており、来店前・来店中・来店後を一貫した顧客体験として設計するという視点を持つプレイヤーがいない。

**空白地帯②：「誰がやっても再現できる」ことを明確に訴求するサービス**  
「仕組み化」「マニュアル作成」は多くの会社が提供するが、「スタッフが変わっても同じ顧客体験が提供できる設計」という再現性を前面に出しているプレイヤーは確認できない。

**空白地帯③：物販・LTVを顧客体験の中に組み込んで設計するコンサル**  
物販促進・LTV向上は別々の施策として語られることが多く、「顧客体験の中に物販や次回予約を自然に埋め込む設計」として提供されているサービスは見当たらない。

**空白地帯④：個人コンサルタントによるCX設計専門サービス**  
個人コンサルは価格の安さやフットワークの軽さが強みだが、そのほぼ全員が「集客」「SNS」「リピート対策」に集中しており、CX・仕組み設計を専門とする個人コンサルは存在しない（推測）。

---

### ポジショニングの考察

以下の2軸でポジショニングマップを整理すると、辻のサービスは競合なしの空白に近い位置を取れる可能性がある。

```
【縦軸】介入範囲：点（個別施策）↔ 面（体験全体の設計）
【横軸】対象フェーズ：新規集客↔ 既存顧客のLTV最大化

・集客コンサル・SNSコンサル → 左下（新規集客の点施策）
・船井総研・フューチャーブレーン → 中央（幅広いが高価格・大企業向け）
・b-models → 中央やや右（カウンセリング設計は強いが体験全体ではない）
・辻のポジション候補 → 右上（既存顧客を中心に、顧客体験全体を面として設計）
```

**参入にあたっての強みとなる要素**
- 「誰がやっても同じ成果が出る」という再現性設計の思想（競合が打ち出していない）
- カスタマージャーニー・LTV設計・クロスセル/アップセルを統合した設計提供
- 実際のサロン（tela filo等）での実績に基づく現場リアリティ
- 個人コンサルとしての低〜中価格帯・高速PDCAのフットワーク

**想定される価格帯**
- 大手（月15万〜25万）と個人集客系（月3万〜10万）の間に空白がある
- 月額8万〜15万円程度の「個人コンサル×CX設計専門」という価格帯は未開拓（推測）
