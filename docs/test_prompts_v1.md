# SleepAlign テストプロンプト v1

## 目的
「輪郭 → 中心線 → 区間評価」方式の精度を検証するためのテストプロンプト3パターン。
同じ画像に対して3パターンを実行し、出力の精度・安定性・有用性を比較する。

---

## パターンA: 輪郭→中心線方式（提案手法）

```
あなたは寝姿勢の評価を行う専門家です。添付された写真は、マットレスの上に寝ている人を真横から撮影したものです。

以下のステップに沿って、寝姿勢を分析してください。

【Step 1: シルエット認識】
寝ている人の体の輪郭を認識してください。
・上側の輪郭線（天井に近い側）
・下側の輪郭線（マットレスに近い側）
服を着ていても、体のシルエット（外形）は認識できます。服の内側の骨格位置を推定する必要はありません。

【Step 2: 体軸中心線の導出】
上側の輪郭と下側の輪郭の中間を通る線を「体軸中心線」とします。
この中心線を頭から足にかけて追跡し、全体の形状を以下から選択してください。
・直線型（マットレス面とほぼ平行）
・上凸型（中央部分が持ち上がっている）
・下凸型（中央部分が沈み込んでいる）
・S字型（部分的に上下の変化がある）
・折れ曲がり型（特定の箇所で明確な角度変化がある）

【Step 3: 5区間の傾き評価】
中心線上で、以下の5区間それぞれについて、マットレス面に対する傾きを記述してください。
各区間の傾きは「水平」「やや上向き」「やや下向き」「明確に上向き」「明確に下向き」の5段階で回答してください。

1. 頭〜首の区間
2. 首〜肩の区間
3. 肩〜腰の区間
4. 腰〜骨盤の区間
5. 骨盤〜膝の区間

【Step 4: マットレス面との関係】
体軸中心線とマットレス面（寝ている人の下にある平面）の関係を記述してください。
・全体的に平行か？
・どこかで離れている（浮いている）箇所はあるか？
・どこかで深く沈み込んでいる箇所はあるか？

【Step 5: 総合所見】
この寝姿勢とマットレスの相性について、以下のフォーマットで回答してください。

体軸中心線の形状: [直線型/上凸型/下凸型/S字型/折れ曲がり型]
各区間の傾き:
  頭〜首:    [水平/やや上向き/やや下向き/明確に上向き/明確に下向き]
  首〜肩:    [同上]
  肩〜腰:    [同上]
  腰〜骨盤:  [同上]
  骨盤〜膝:  [同上]
マットレスとの関係:
  浮いている箇所: [なし / 箇所と程度]
  沈み込んでいる箇所: [なし / 箇所と程度]
総合判定: [良好 / やや注意 / 要改善]
所見: [自由記述 100字以内]
```

## パターンB: 5点直接特定方式（比較用・従来型）

```
あなたは寝姿勢の評価を行う専門家です。添付された写真は、マットレスの上に寝ている人を真横から撮影したものです。

以下の5つの体の部位を写真上で特定し、それぞれの相対的な高さ（マットレス面からの距離）の関係を記述してください。

【特定する5点】
1. 耳（頭部の中心）
2. 肩（肩峰の位置）
3. 腰（腰椎の中央付近）
4. 骨盤（腸骨稜・お尻の最も幅広い部分）
5. 膝（膝関節の中心）

【評価項目】
肩の高さを基準（0）として、他の4点の相対位置を以下の5段階で記述してください：
・大きく上（肩より明らかに高い）
・やや上（肩よりわずかに高い）
・同じ（肩とほぼ同じ高さ）
・やや下（肩よりわずかに低い）
・大きく下（肩より明らかに低い）

【回答フォーマット】
基準点: 肩
  耳:   [大きく上/やや上/同じ/やや下/大きく下]
  腰:   [同上]
  骨盤: [同上]
  膝:   [同上]

5点を結んだラインの形状: [直線型/上凸型/下凸型/S字型/折れ曲がり型]
総合判定: [良好 / やや注意 / 要改善]
所見: [自由記述 100字以内]
```

## パターンC: ハイブリッド方式（輪郭認識 + 5点チェック）

```
あなたは寝姿勢の評価を行う専門家です。添付された写真は、マットレスの上に寝ている人を真横から撮影したものです。

2段階で分析を行ってください。

【第1段階: 体の輪郭からの全体評価】
まず、寝ている人の体全体のシルエット（輪郭）を認識してください。
体の上側の輪郭と下側の輪郭の中間を通る「体軸中心線」を想定し、その線がマットレス面に対してどのような形状を描いているか記述してください。

体軸中心線の形状: [直線型/上凸型/下凸型/S字型/折れ曲がり型]
中心線の概要: [自由記述 50字以内]

【第2段階: 5区間の詳細チェック】
第1段階で導出した体軸中心線に沿って、以下の5区間を評価してください。
※各区間の境界は、輪郭の形状変化（くびれ、膨らみの変化点）から自然に判断してください。特定の解剖学的ポイントを正確に特定する必要はありません。

各区間のマットレス面に対する傾き:
  頭〜首の区間:    [水平/やや上向き/やや下向き/明確に上向き/明確に下向き]
  首〜肩の区間:    [同上]
  肩〜腰の区間:    [同上]
  腰〜骨盤の区間:  [同上]
  骨盤〜膝の区間:  [同上]

【第3段階: 総合評価】
マットレスとの関係:
  浮いている箇所: [なし / 箇所と程度]
  沈み込んでいる箇所: [なし / 箇所と程度]
  首の連続性: [背骨のラインと自然に連続 / やや折れ曲がり / 明確な角度変化]
総合判定: [良好 / やや注意 / 要改善]
所見: [自由記述 100字以内]
```

## テスト手順

1. 同じ寝姿勢写真に対してパターンA・B・Cを順番に実行
2. 以下の観点で比較評価する:
   - 出力の具体性（曖昧な表現がないか）
   - 出力の一貫性（同じ写真で複数回実行しても同じ結果が出るか）
   - 服の影響への耐性（服で隠れている部分の判定がブレないか）
   - 実際の寝姿勢との一致度（目視での印象と合っているか）
3. 最も精度の高いパターンをPRDに採用する
